1. git init
    1.1. git config --global user.email "malripo711205@gmail.com" 
    1.2. git config --global user.name "malripo"
2. git add forder명/ or 파일명
   git add Notice/
3. git commit -m "init_001"  <= 의미있는 커밋 내용
4. git remote add origin (admin.txt의 주소)) : github와 local 연결
   git remote add origin https://github.com/malripo/TIL.git
5. git push origin master
6. 참고 명령어
   6-1). git status
   6-2). git log --oneline     : 한 줄로 축약해서 보여줍니다.
         git log --oneline     : 한 줄로 축약해서 보여줍니다.
         git log --graph       : 브랜치와 머지 내역을 그래프로 보여줍니다.
         git log --all         : 현재 브랜치를 포함한 모든 브랜치의 내역을 보여줍니다.
         git log --reverse     : 커밋 내역의 순서를 반대로 보여줍니다. (최신이 가장 아래)
         git log -p            : 파일의 변경 내용도 같이 보여줍니다.
         git log -2            : 원하는 갯수 만큼의 내역을 보여줍니다. (2 말고 임의의 숫자 사용 가능)
   6-3). git remote -v
   6-4). git config --global --list
   6-5). git remote rm origin 으로 제거 가능
         git remote remove origin
